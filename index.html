LiDAR Visualization Tool

LiDAR Visualization Tool (LVT) is an open-source application enabling autonomous vehicle researchers to visualize data from different formats and gives them a comprehensive tool to analyze their machine learning algorithms

Features:

LCT Data Format API

  • camera: contains the RGB images from different cameras
  • pointcloud: contains PCD files with LiDAR pointclouds for each sensor
  • bounding: contains the ground truth bounding boxes for each frame as JSON files
  • pred_bounding: contains the predicted bounding boxes for each frame as JSON files
  • ego: contains the translation and rotation for each frame as JSON files

Conversion Tool

Utility functions provided to aid in converting any dataset into the LCT data format

Built-in nuScenes and Waymo conversion script

Visualization Tool

Visualizes LCT data format

Easily switch between point cloud and RGB view

Rotate point cloud in free space

Filter what annotations are shown
